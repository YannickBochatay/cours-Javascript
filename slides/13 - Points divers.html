Introductions à ES6

  • Fonctions fléchées : raccourcis pour fonctions anonymes
  • Portée de bloc avec l'opérateur let
  • Constantes avec le mot clé const
  • Classes : sucre syntaxique pour l'héritage prototypal
  • Gabarits de chaînes de caractères (template strings)
  • Paramètres par défaut, paramètres du reste, opérateur de décomposition
  • Affectation par décomposition pour les tableaux et objets
  • Itérateurs et Générateurs
  • Map, Set, WeakMap, WeakSet : structures de données
  • Modules pour organiser et charger le code
  • Promises pour gérer l'asynchronisme
