Nombres, chaînes de caractères et booléens

Valeurs primitives ou objets ?

Nous avons vu que :

  • les nombres, chaînes de caractères et booléens sont des types primitifs de javascript
  • ce qui n'est pas de type primitif est une variation du type objet
  • il existe des constructeurs Number, String, Boolean.
var age = 25; //age est-il un nombre ou un objet nombre ?
					
var nom = "toto"; //nom est-il une chaîne ou un objet chaîne ?

var bool = true; //bool est-il un booléen ou un objet booléen ?

Valeurs primitives ou objets ?

Ce sont bien des valeurs primitives que nous manipulons :

var obj = {};
typeof obj; //object

var age = 25;
typeof age; //number
					
var nom = "toto";
typeof nom; //string

var bool = true;
typeof bool; //boolean

Valeurs primitives ou objets ?

Mais alors, pourquoi pouvons-nous faire ceci ?

var age = 25;
age.toString(); //"25"
					
var nom = "toto";
nom.replace('o','a'); //tato

var bool = true;
bool.toString(); //true

Javascript effectue automatiquement les conversions entre valeurs primitives et objets.

Valeurs primitives ou objets ?

On instancie très rarement des objets à partir des constructeurs String, Boolean, Number.

var age = new Number(25);
var nom = new String("toto");                    //très rare
var bool = new Boolean(true);

➨ plus lourd et peut conduire à des pièges :

var nom = new String("toto");
typeof nom; //"object" et non "string"

var nom2 = new String("toto");
nom == nom2; //false (deux instances ne sont jamais égales)

nom.valueOf() == nom2.valueOf(); //true
//la méthode valueOf permet de convertir en type primitif

Valeurs primitives ou objets ?

En revanche l'invocation des constructeurs String, Boolean, Number sans le mot clef new peut s'avérer utile, elle convertit alors l'argument en valeur primitive correspondante.

Number("25.5"); //25.5
Number("25.5Ko"); //NaN
Number(true); //1
Number(false); //0

String(25.5); //"25.5"
String(true); //"true"
String(false); //"false"

Boolean("toto"); //true
Boolean(""); //false
Boolean(25); //true
Boolean(0); //false

Méthodes usuelles des chaînes

  • charAt : renvoie le caractère à la position spécifiée
  • indexOf : renvoie l'indice de la 1ère occurence
  • lastIndexOf : renvoie l'indice de la dernière occurence
  • replace : renvoie la chaîne modifiée à l'aide d'une expression rationnelle et un texte de remplacement
  • search : recherche grâce à une expression rationnelle
  • slice : extrait une section de la chaine de caractères
  • split : divise la chaîne en tableau de sous-chaînes
  • toLowerCase : renvoie la chaîne en minuscules
  • toUpperCase : renvoie la chaîne en majuscules
  • trim : renvoie la chaîne en retirant les blancs en début et fin

Méthodes usuelles des chaînes

Toutes ces méthodes ne modifient pas la chaîne elle-même mais renvoient une nouvelle valeur primitive de type "string".

var nom = "Toto";
nom.replace('o','a');
console.log(nom); //Toto
						
var nomModifie = nom.replace('o','a');
console.log( nomModifie ); //Tato
