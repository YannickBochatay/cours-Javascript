Bouton haut de page

Le but de cet exercice est de créer un bouton "Retour en haut de page" lorsqu'on descend à plus de 500px du haut de page. Dans le cas contraire, le bouton est masqué.
En cliquant sur le bouton, on revient en haut de page (fonctionnalité déjà présente).

Eléments à utiliser :

  • l'évènement scroll déclenché par l'objet window
  • la méthode addEventListener de l'objet window
  • la propriété scrollY de l'objet window
  • la propriété classList des éléments HTML
  • la classe css "hidden" définie dans le fichier styles.css
